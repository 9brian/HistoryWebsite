What is it?

Java is one of the popular programming languages out there.
It's known to be an Object-Oriented programming language and used by many across digital devices.
Where you can see java in the real world is on Android apps, web applications, software tools and many more places!


History of Java

Java, released in the early 1990s was created and developed by James Gosling.
Initially, the project to create a language that can span across digital devices was one James took head on.
C++ was the initial language to take on this project, it was experimented on and expanded.
Java took on many names before it's famous name known to day, such as Greentalk, and Oak.
Later being changed to Java, inspired by the espresso bean and Gosling's cup of coffee.
